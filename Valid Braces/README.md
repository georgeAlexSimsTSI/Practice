# Valid-Braces
Function that takes a string of braces and determines if the order of braces is valid.
<br> e.g. </br>


| Input  | Output |
| ------------- |:-------------:|
| "(){}[]"      | True          |
| "([{}])"      | True          |
| "(}"          | False         |
| "[(])"        | False         |
| "[({})](]"    | False         |
